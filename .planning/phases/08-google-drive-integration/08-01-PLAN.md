    Functions (all with lazy imports of google libraries inside function body):

    - `get_token_path(config_dir: Path) -> Path`: Returns `config_dir / "google_token.json"` per D-02.

    - `get_credentials(token_path: Path) -> Credentials | None`: If file missing, return None. Load via `Credentials.from_authorized_user_file(str(token_path), SCOPES)`. If expired and has refresh_token, call `creds.refresh(Request())`, then call `_save_token(token_path, creds, folder_id=_load_folder_id(token_path))` to persist refreshed creds while preserving cached folder_id. Return creds if valid, else None.

    - `run_auth_flow(token_path: Path) -> Credentials`: Call `InstalledAppFlow.from_client_config(CLIENT_CONFIG, scopes=SCOPES)`, then `flow.run_local_server(port=0, access_type="offline", prompt="consent")` per D-03. Write token via `_save_token(token_path, creds)`. Return creds.

    - `build_service(creds)`: Lazy import `from googleapiclient.discovery import build`. Return `build("drive", "v3", credentials=creds)`.

    - `get_or_create_folder(service, folder_name: str) -> str`: Query `service.files().list(q="name='{folder_name}' and mimeType='application/vnd.google-apps.folder' and trashed=false", spaces="drive", fields="files(id,name)")`. If results, return `folders[0]["id"]`. Else create with `service.files().create(body={"name": folder_name, "mimeType": "application/vnd.google-apps.folder"}, fields="id")` and return the id.

    - `upload_file(service, local_path: Path, folder_id: str) -> str`: Lazy import `from googleapiclient.http import MediaFileUpload`. Determine mime from `MIME_TYPES.get(local_path.suffix.lstrip("."), "text/plain")`. Create metadata `{"name": local_path.name, "parents": [folder_id]}`. Call `service.files().create(body=metadata, media_body=MediaFileUpload(str(local_path), mimetype=mime), fields="id").execute()`. Return file id.

    - `_save_token(token_path: Path, creds, folder_id: str | None = None) -> None`: Build dict from `json.loads(creds.to_json())`. If folder_id, add `data["drive_folder_id"] = folder_id`. Write JSON to token_path, then `token_path.chmod(0o600)`.

    - `_load_folder_id(token_path: Path) -> str | None`: If file missing, return None. Parse JSON, return `data.get("drive_folder_id")`.

    - `upload_transcripts(config_dir: Path, files: list[Path], folder_name: str) -> None`: Load creds via `get_credentials(get_token_path(config_dir))`. If None, raise `RuntimeError("Not authenticated with Google Drive. Run: mote auth google")`. Build service. Load cached folder_id via `_load_folder_id`. If no cached id, call `get_or_create_folder(service, folder_name)` and persist via `_save_token(token_path, creds, folder_id=folder_id)`. Upload each file via `upload_file(service, f, folder_id)`.
