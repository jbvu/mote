    **1. Create src/mote/notebooklm.py** (per D-02, D-03, D-06, D-07):

    Module docstring: `"""NotebookLM API wrapper for Mote transcript uploads (experimental)."""`

    Imports at top (non-lazy): `import asyncio`, `import json`, `from pathlib import Path`

    Constants:
    - `SESSION_FILE = "notebooklm_session.json"`

    Functions (all function-based, no classes):

    `get_session_path(config_dir: Path) -> Path`: return `config_dir / SESSION_FILE`

    `is_authenticated(config_dir: Path) -> bool`: return `get_session_path(config_dir).exists()`

    `run_login(session_path: Path) -> None`:
    - Lazy import: `import subprocess`
    - Run: `subprocess.run(["notebooklm", "login", "--storage", str(session_path)], check=False)`
    - If returncode != 0: raise `RuntimeError("NotebookLM login failed. Try running 'notebooklm login' manually.")`
    - Set `session_path.chmod(0o600)`

    `_load_notebook_id(session_path: Path) -> str | None`:
    - If not exists, return None
    - Try parse JSON, return `data.get("notebook_id")`
    - On JSONDecodeError/OSError, return None

    `_save_notebook_id(session_path: Path, notebook_id: str) -> None`:
    - Read existing JSON from session_path
    - Set `data["notebook_id"] = notebook_id`
    - Write back, chmod 0o600

    `async def _get_or_create_notebook(client, notebook_name: str) -> str`:
    - `notebooks = await client.notebooks.list()`
    - Loop: if `nb.title == notebook_name`, return `nb.id`
    - Else: `nb = await client.notebooks.create(notebook_name)`, return `nb.id`

    `async def _upload_async(session_path: Path, notebook_name: str, title: str, content: str) -> None`:
    - Lazy import: `from notebooklm import NotebookLMClient`
    - `async with await NotebookLMClient.from_storage(str(session_path)) as client:`
    - Load cached notebook_id via `_load_notebook_id(session_path)`
    - If None: call `_get_or_create_notebook(client, notebook_name)`, save with `_save_notebook_id`
    - Try `await client.sources.add_text(notebook_id, title, content)`
    - On Exception: invalidate cache, call `_get_or_create_notebook` again, save, retry `add_text`

    `upload_transcript(config_dir: Path, files: list[Path], notebook_name: str) -> None`:
    - Get session_path, check exists, raise RuntimeError("Not authenticated with NotebookLM. Run: mote auth notebooklm") if not
    - Filter `md_files = [f for f in files if f.suffix == ".md"]`
    - If not md_files: return (silent no-op per D-06)
    - For each md_file: `title = md_file.stem`, `content = md_file.read_text()`, `asyncio.run(_upload_async(session_path, notebook_name, title, content))`
