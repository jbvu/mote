  <action>
    Create src/mote/output.py as a function-based module (no classes, per project convention):

    1. **_HEADER_TEMPLATE** constant (per D-01):
       ```
       ---
       date: {date}
       duration: {duration}
       words: {words}
       engine: {engine}
       language: {language}
       model: {model}
       ---

       {transcript}
       ```
       - `date` = `ts.isoformat()` (ISO 8601)
       - `duration` = `round(duration_seconds)` (integer seconds)
       - `words` = `len(transcript.split())`

    2. **_build_filename(ts: datetime, name: str | None, ext: str) -> str**: `ts.strftime("%Y-%m-%d_%H%M")` + optional `_{name}` + `.{ext}` (per D-05)

    3. **_sanitize_name(raw: str) -> str** (per D-07):
       - `raw.lower().strip()`
       - `re.sub(r"\s+", "-", s)`
       - `re.sub(r"[^a-z0-9-]", "", s)`
       - `re.sub(r"-{2,}", "-", s)`
       - `.strip("-")`

    4. **write_transcript(transcript, output_dir, formats, duration_seconds, engine, language, model_alias, name=None, timestamp=None) -> list[Path]** (per D-13, D-15):
       - `ts = timestamp or datetime.now()`
       - `output_dir.mkdir(parents=True, exist_ok=True)` (per D-08)
       - If "markdown" in formats: write .md with _HEADER_TEMPLATE
       - If "txt" in formats: write .txt with transcript text only (per D-02)
       - Return list of written Path objects

    5. **_HEADER_RE** regex constant for parsing:
       ```python
       _HEADER_RE = re.compile(
           r"^---\n"
           r"date: (?P<date>[^\n]+)\n"
           r"duration: (?P<duration>\d+)\n"
           r"words: (?P<words>\d+)\n"
           r"engine: (?P<engine>[^\n]+)\n"
           r"language: (?P<language>[^\n]+)\n"
           r"model: (?P<model>[^\n]+)\n"
           r"---",
       )
       ```

    6. **list_transcripts(output_dir: Path) -> list[dict]** (per D-12):
       - Return [] if dir doesn't exist
       - Glob `*.md`, sort by `stat().st_mtime` descending (newest first)
       - For each file, read text, match _HEADER_RE, skip on no match or OSError
       - Return list of dicts: {"filename": str, "date": str, "duration": int, "words": int, "engine": str}
