    Then implement src/mote/audio.py with these functions:

    1. `find_blackhole_device() -> Optional[dict]` (per D-07, D-08):
       - Call `sd.query_devices()`, iterate devices
       - Match devices where `"BlackHole" in d["name"]` AND `d["max_input_channels"] > 0`
       - Prefer device with `"2ch" in d["name"]`; fall back to any other BlackHole variant
       - Return the device dict or None

    2. `rms_db(data: np.ndarray) -> float`:
       - Compute `float(np.sqrt(np.mean(data ** 2)))`
       - Return `max(-60.0, 20.0 * np.log10(max(rms, 1e-9)))`
       - Floor at -60.0 dB (not just 1e-9 limit — explicitly clamp return value)

    3. `make_level_bar(db: float, width: int = 20) -> str`:
       - Map -60dB..0dB to 0..width
       - `level = max(0.0, min(1.0, (db + 60.0) / 60.0))`
       - `filled = int(level * width)`
       - Return `"█" * filled + "░" * (width - filled)`

    4. `make_display(elapsed_s: int, db: float) -> Text` (per D-01):
       - Format elapsed as HH:MM:SS: `hours, rem = divmod(elapsed_s, 3600); mm, ss = divmod(rem, 60)`
       - Build Rich `Text` object: "Recording  HH:MM:SS  {bar}  {db}dB"
       - Use styles: `"bold cyan"` for time, `"green"` for bar, `"dim"` for dB

    5. `write_wav(path: Path, chunks: list[np.ndarray], samplerate: int = 16000) -> None` (per D-06):
       - `audio = np.concatenate(chunks, axis=0)`
       - `audio_int16 = (audio * 32767).clip(-32768, 32767).astype(np.int16)` (per Pitfall 5)
       - Use stdlib `wave.open(str(path), "wb")` with nchannels=1, sampwidth=2, framerate=samplerate

    6. `new_recording_path(recordings_dir: Path) -> Path` (per D-04):
       - `recordings_dir.mkdir(parents=True, exist_ok=True)`
       - Filename: `mote_YYYYMMDD_HHMMSS.wav` using `datetime.datetime.now().strftime("%Y%m%d_%H%M%S")`

    7. `is_recording_active(pid_path: Path) -> tuple[bool, Optional[int]]` (per D-10, D-11, D-12):
       - If pid_path doesn't exist: return (False, None)
       - Read PID, catch ValueError for corrupt files → (False, None)
       - `os.kill(pid, 0)` — ProcessLookupError → (False, pid) [stale], PermissionError → (True, pid), success → (True, pid)

    8. `find_orphan_recordings(recordings_dir: Path) -> list[Path]` (per D-05):
       - Return `sorted(recordings_dir.glob("*.wav"))` if dir exists, else empty list

    9. `record_session(device_index: int, recordings_dir: Path, pid_path: Path) -> Path` (per AUD-01, AUD-02, AUD-03, AUD-04):
       - Constants: SAMPLE_RATE=16000, CHANNELS=1, BLOCKSIZE=1024
       - Create queue.Queue, threading.Event (stop_event), chunks list
       - Define `_audio_callback(indata, frames, time, status)` that calls `audio_queue.put(indata.copy())`
       - Set `signal.signal(signal.SIGINT, lambda s, f: stop_event.set())`
       - Write PID file: `pid_path.write_text(str(os.getpid()))`
       - Create wav_path via `new_recording_path(recordings_dir)`
       - Print device info once at start (per D-02): `"Recording from {device_name} (16kHz mono)"`
       - Open `sd.InputStream(samplerate=SAMPLE_RATE, channels=CHANNELS, dtype="float32", blocksize=BLOCKSIZE, device=device_index, callback=_audio_callback)`
       - Inside Rich `Live(make_display(0, -60.0), refresh_per_second=4)` context (per D-01):
         - Main loop: while not stop_event.is_set(), get chunk with timeout=0.1 (per Pitfall 6), append to chunks, compute dB, update display
         - Below the live line show "Ctrl+C to stop" (per D-03) — use `console.print` before entering Live, or include in display Text
       - In finally block: delete PID file, write WAV via write_wav if chunks not empty
       - Return wav_path
